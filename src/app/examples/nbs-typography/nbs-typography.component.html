Brand typography

The proper use of headings is important for search engines which use them to index the structure and content of pages.

Furthermore, they serve to improve the readability of often complex interfaces, visually setting out the structure of page content and making them easier to read for users.

Typography in Angular

The nbs-material.theme file included in this project sets up all of the core typographical elements, additional classes are provided in the base > typography file to supliment the theme and provide a greater diversity of options.

Azo Sans

Azo Sans has been chosen as our primary typeface for its modernity, simplicity and its close visual link to the NBS logo.

Our websites should use the Azo Sans Web variant and use the font-weight CSS rule to adjust the weight of the font.

Acceptable alternatives

When Azo Sans Web is not available you should use Arial as the fallback font.

Colour, case, justification, etc.

Headlines, titles and callouts may appear in NBS Primary. Small annotations may also appear in NBS Secondary. Body copy should appear in NBS Primary Text.

Sentence case is standard for all communications. Try to avoid using all uppercase wherever possible.

Type should always be aligned left. Avoid lines that end with hyphens, widows and orphans.

Italics can be used in special projects on a case by case basis.
