Menus

Menus display a list of choices on temporary surfaces. They appear when users interact with a button, action, or other control.

Principles

Nimble - Menus should be easy to open, close, and interact with.

Contextual - Menu content should be suited to user needs.

Scannable - Menu items should be easy to scan.
