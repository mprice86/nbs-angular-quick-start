Brand form field examples

Forms and the inputs they contain allow the user to enter text into the UI.

Our branded form inputs are a custom version of the Material Design Outline style inputs. These inputs have been written to deliver specific styling and interaction set out in our UX Component documentation. If you adopt this style for your site it should be applied across the board with all inputs following the style.

Form inputs are required to be wrapped in a containing element with the class 'form-control-wrapper'. Within the 'form-control-wrapper' you should provide a label (where appropriate, see examples) and the material version of the form field you want to use.

The example provided shows the ideal spacing between form fields, as well as demonstrating how you could validate your form and provide feedback to the user.

Guidance

For more information on the Angular implementation of forms, see the Angular Material form field article.

For more information on the Material Design principles of form fields see the Material Design text fields article
